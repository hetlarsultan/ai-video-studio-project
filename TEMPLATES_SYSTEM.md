# نظام مكتبة القوالب - AI Video Studio

## نظرة عامة

تم إضافة نظام متكامل لإدارة قوالب الفيديو الجاهزة في تطبيق AI Video Studio. يوفر هذا النظام مكتبة شاملة من القوالب المتنوعة لتسريع عملية إنشاء الفيديو.

## المكونات الرئيسية

### 1. قاعدة البيانات (Database)

#### جدول `templates`
يحتوي على جميع معلومات القوالب:

| الحقل | النوع | الوصف |
|------|------|-------|
| `id` | INT | معرف فريد للقالب (مفتاح أساسي) |
| `title` | VARCHAR(255) | عنوان القالب |
| `description` | TEXT | وصف تفصيلي للقالب |
| `thumbnailUrl` | VARCHAR(512) | رابط صورة معاينة القالب |
| `category` | VARCHAR(100) | فئة القالب (وسائل اجتماعية، تعليمي، إلخ) |
| `config` | TEXT | إعدادات القالب بصيغة JSON |
| `isPremium` | INT | 0 للقوالب المجانية، 1 للقوالب المتميزة |
| `usageCount` | INT | عدد مرات استخدام القالب |
| `createdAt` | TIMESTAMP | تاريخ إنشاء القالب |
| `updatedAt` | TIMESTAMP | تاريخ آخر تحديث |

### 2. خدمات قاعدة البيانات (db.ts)

تم إضافة الدوال التالية:

```typescript
// الحصول على جميع القوالب
getTemplates(): Promise<Template[]>

// الحصول على قالب محدد برقمه
getTemplateById(id: number): Promise<Template | null>

// زيادة عدد مرات استخدام القالب
incrementTemplateUsage(id: number): Promise<void>

// إضافة قالب جديد
insertTemplate(template: InsertTemplate): Promise<void>
```

### 3. واجهة برمجية (API) - templateRouter.ts

#### الإجراءات المتاحة:

**1. `listTemplates`** - الحصول على قائمة القوالب
```typescript
Input: {
  category?: string;        // اختياري: تصفية حسب الفئة
  includePremium?: boolean; // اختياري: تضمين القوالب المتميزة
}

Output: {
  success: boolean;
  templates: Template[];
  count: number;
}
```

**2. `getTemplate`** - الحصول على تفاصيل قالب محدد
```typescript
Input: {
  id: number; // معرف القالب
}

Output: {
  success: boolean;
  template: TemplateDetails | null;
}
```

**3. `getTemplatesByCategory`** - الحصول على القوالب حسب الفئة
```typescript
Input: {
  category: string; // اسم الفئة
}

Output: {
  success: boolean;
  templates: Template[];
  count: number;
}
```

**4. `getCategories`** - الحصول على قائمة الفئات المتاحة
```typescript
Output: {
  success: boolean;
  categories: string[];
}
```

**5. `getPopularTemplates`** - الحصول على القوالب الأكثر استخداماً
```typescript
Input: {
  limit?: number; // عدد القوالب (افتراضي: 10)
}

Output: {
  success: boolean;
  templates: Template[];
}
```

### 4. مكونات واجهة المستخدم

- معاينة سريعة لكل قالب

#### `TemplatesPage.tsx`
صفحة متخصصة تحتوي على:

### 5. بيانات القوالب الجاهزة

تم إنشاء 16 قالب جاهز موزعة على الفئات التالية:

#### وسائل اجتماعية (3 قوالب)
- فيديو تيك توك احترافي
- منشور إنستاجرام ريلز
- فيديو يوتيوب شورتس

#### تعليمي (3 قوالب)
- شرح تعليمي مفصل
- درس تفاعلي (متميز)
- عرض شرائح تقديمي

#### تسويق (3 قوالب)
- إعلان منتج احترافي (متميز)
- فيديو دعاية الخدمة
- فيديو شهادات العملاء

#### ترفيه (3 قوالب)
- مقدمة فيديو احترافية (متميز)
- فيديو موسيقي (متميز)
- فيديو كوميدي

#### رسوميات (2 قالب)
- رسوميات متحركة احترافية (متميز)
- شرح رسومي (Explainer)

#### شركات (2 قالب)
- فيديو شركة احترافي (متميز)
- فيديو تعريفي للشركة

#### أحداث (2 قالب)
- فيديو حفل افتتاح (متميز)
- فيديو ملخص الحدث

## كيفية الاستخدام

### 1. تشغيل البذر (Seeding)

لملء قاعدة البيانات بالقوالب الجاهزة:

```bash
npm run db:seed
# أو
pnpm db:seed
```

### 2. الوصول إلى مكتبة القوالب

**من الواجهة:**
- انتقل إلى `/templates` في التطبيق

**من الـ API:**
```typescript
// الحصول على جميع القوالب
const templates = await trpc.templates.listTemplates.query({
  category: 'وسائل اجتماعية'
});

// الحصول على قالب محدد
const template = await trpc.templates.getTemplate.query({
  id: 1
});
```

### 3. استخدام قالب

عند اختيار قالب:
1. يتم تحميل تفاصيل القالب
2. يتم زيادة عدد مرات الاستخدام تلقائياً
3. يمكن نسخ الإعدادات أو استخدام القالب مباشرة
4. يتم حفظ إعدادات القالب في `localStorage`

## هيكل إعدادات القالب (Config)

كل قالب يحتوي على إعدادات JSON تحدد خصائصه:

```json
{
  "duration": 30,
  "resolution": "1080x1920",
  "fps": 30,
  "effects": ["fade", "zoom"],
  "music": true,
  "voiceover": false,
  "subtitles": true,
  "style": "modern"
}
```

### الحقول المتاحة:
- `duration`: مدة الفيديو بالثواني
- `resolution`: دقة الفيديو (مثل 1920x1080)
- `fps`: عدد الإطارات في الثانية
- `effects`: مصفوفة التأثيرات المتاحة
- `music`: وجود موسيقى خلفية
- `voiceover`: وجود تعليق صوتي
- `subtitles`: وجود ترجمة نصية
- `style`: نمط القالب

## التوسع والتطوير

### إضافة قوالب جديدة

1. أضف القالب إلى `PREDEFINED_TEMPLATES` في `server/seedTemplates.ts`
2. قم بتشغيل `npm run db:seed`

### تعديل القوالب الموجودة

يمكن تعديل القوالب مباشرة في قاعدة البيانات أو إنشاء واجهة إدارة متقدمة.

### إضافة فئات جديدة

ما عليك سوى إضافة قوالب بفئات جديدة، سيتم اكتشافها تلقائياً.

## الميزات الإضافية

### تتبع الاستخدام
- يتم تسجيل عدد مرات استخدام كل قالب
- يمكن استخدام هذه البيانات لتحديد القوالب الشهيرة

### القوالب المتميزة
- يمكن تحديد قوالب كمتميزة (isPremium = 1)
- يمكن إضافة منطق للتحقق من الاشتراك قبل السماح باستخدامها

### التصفية والبحث
- تصفية حسب الفئة
- تصفية حسب القوالب المتميزة
- عرض القوالب الأكثر استخداماً

## الخطوات التالية

1. **تحسين الواجهة**: إضافة مزيد من خيارات التصفية والبحث
2. **إدارة القوالب**: إنشاء واجهة إدارة لإضافة وتعديل القوالب
3. **التكامل مع المحرر**: ربط القوالب مع محرر الفيديو الرئيسي
4. **التحليلات**: إضافة تحليلات متقدمة لاستخدام القوالب
5. **المشاركة**: السماح للمستخدمين بمشاركة قوالبهم الخاصة
